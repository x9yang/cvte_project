# 法务合同分派智能体

## 课题名称

法务合同分派智能体设计与实现

---

# 一、项目背景

公司法务团队每天需要处理大量合同审批任务。

目前合同进入审批流程后，需要由人工根据合同内容、合同类型以及法务人员的专业能力进行任务分派。

随着业务增长，合同数量不断增加，人工分派存在以下问题：

* 分派效率低
* 依赖经验判断
* 工作量统计困难
* 分派结果缺乏统一标准
* 难以保证负载均衡

因此，希望利用大模型及Agent技术，探索合同智能分派场景的落地方案。

---

# 二、智能体分派计算维度

### 输入维度（举例）

| 类别 | 数据字段 | 说明 |
|------|----------|------|
| 合同静态特征 | 页数、总字数 | 用于估算工作量（处理时间） |
| 合同动态特征 | 合同类型（分类） | 经大模型解析得到：租赁、筹建、购销 等 |
| 法务人员能力 | 技能标签 | 与合同类型匹配，例如：<br>• 张三 → 租赁合同<br>• 李四 → 购销合同 |
| 法务人员负载 | • 当前待办数量<br>• 本周已分配数量<br>• 本月已分配数量 | 用于负载均衡，支持不同时间粒度 |

### 输出
- **推荐人选**（单个法务人员姓名）

### 扩展性要求
- 合同类型可动态增加（如劳动、保密、知识产权等），不应硬编码
- 法务人员的技能标签支持多对多（一人擅长多种类型）
- 负载维度可扩展（未来可加入平均处理时长、紧急任务数等）
- 优先级机制可配置（当前合同类型匹配为**优先级1**，后续可加入风险等级优先级2等）
